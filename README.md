# internship-customer-power-safety-system
## 项目简介
实习实训项目——电力运维维修管理系统静态介绍页面，基于原生 HTML + CSS + JavaScript 开发，用于完整展示一套**客户安全用电运维检修管理系统**业务架构、流程、功能与双端界面演示。

页面为独立静态网页，无需后端环境，本地打开即可完整浏览，适用于实习作业、课程展示、求职项目演示。

## 项目背景
面向电力客户现场安全运维场景，解决传统线下工单无流程、设备无台账、维修记录无法追溯、派单依靠人工沟通等痛点。
系统分为两大终端：
1. Web 管理后台：管理员调度、设备建档、故障派单、工单审核、数据统计
2. 微信小程序：维修人员现场签到、扫码作业、填写检修表单、电子签名

本介绍页面完整梳理系统价值、全业务闭环流程、技术架构、权限体系与双端界面预览。

## 页面功能模块
1. **首页概览**：双端协同架构可视化展示，快速了解平台整体设计思路
2. **核心能力**：设备一物一码、现场定位校验、智能就近派单三大核心优势
3. **业务闭环流程**：5步工单全流程（故障创建→派单→现场作业→审核签名→归档报告）
4. **系统功能详情**：地图点位、工单流转、现场留痕、电子签章、统计报表、分级权限
5. **数据安全与权限体系**：四级组织权限隔离、接口鉴权、数据脱敏机制
6. **技术架构说明**：小程序 + Vue3管理端 + Flask服务端三层架构
7. **界面演示预览**
   - Web管理端：8张完整功能页面轮播预览
   - 微信小程序端：5张移动端作业页面轮播预览
   - 支持左右切换、底部进度点快速跳转查看截图

## 技术栈（本介绍页面）
- HTML5：页面结构、语义化布局
- CSS3：原生变量样式、弹性/网格布局、响应式适配、渐变阴影动效
- JavaScript：页面滚动导航高亮、Tab切换、图片轮播预览
- 特性：平滑滚动、自适应手机/电脑、hover交互、无障碍语义标签

## 整套业务系统技术栈（页面介绍的真实业务系统）
### 客户端
- 微信小程序：定位、扫码、表单填报、电子签名
- Web管理后台：Vue3 + Vite，地图可视化、工单管理、数据统计

### 服务端
- Python Flask：API接口、JWT身份鉴权、智能派单逻辑
- 工具库：Pillow图像处理、python-docx自动生成维修报告
- 数据库：SQLite 轻量化存储设备、工单、人员档案
